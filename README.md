# homebrew-tap

Homebrew formulae for [sift](https://github.com/raymondj99/sift) — a local semantic search engine with hybrid search, 30+ format indexing, and AI agent memory.

## Install

```bash
brew install raymondj99/tap/sift-search
```

## Usage

```bash
# Index a directory
sift scan ~/my-project

# Search
sift search "error handling patterns"

# Check status
sift status
```

For full documentation, see the [sift README](https://github.com/raymondj99/sift).
